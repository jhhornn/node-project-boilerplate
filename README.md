<h1 align="center">Node(Express.js) Boilerplate</h1>

## Clone this repo

```sh
git clone https://github.com/jhhornn/node-project-boilerplate.git
```

## Install

```sh
npm install
```

## Usage

```sh
npm run dev
```

## Run prettier format

```sh
npm run format
```
